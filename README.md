# speckit-runner-installer

Installer for [speckit-runner](https://github.com/hrbocutti/speckit-runner) — Spec-Driven Development CLI with Claude Code.

## Install

```bash
curl -fsSL https://raw.githubusercontent.com/hrbocutti/speckit-runner-installer/main/install.sh | bash
```

## Prerequisites

- **git**
- **gh** — GitHub CLI, authenticated (`gh auth login`)
- **python3** — 3.9+
- **claude** — Claude Code CLI, configured (`claude doctor`)

## Update

Re-run the same command to update to the latest version.
